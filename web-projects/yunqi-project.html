    # 一、项目介绍

    项目为民宿类的移动 Web 端，用户可以通过移动端进行查看，其中包含新房特惠、房源搜 索、详情、订单、消息等功能。项目主要为游客提供住宿服务，为外出郊游或远行的旅客提供个性化住 宿场所。游客可以通过移动端线上查看房间信息，以及预定房间。

    此文章主要是用于展示项目部分预览图，并非项目开发文档。

    # 二、使用技术

    Vue3、Vite、Pinia、Vue-Router、Axios、Vant-ui、Less
