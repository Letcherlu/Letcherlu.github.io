    # 一、项目介绍

    项目是由 Vue-cli 搭建的 PC 端 SPA 商城，主要用于平台用户的参与积分兑换商品，用户可以在平台上通过做任务、每日签到、购买产品等方式来换取积分，该项目主要涉及登录注册、商品列表、 商品详情、个人中心、购物车及商品搜索等主体功能。

    此文章主要是用于展示项目部分预览图，并非项目开发文档。

    # 二、使用技术

    Vue2、Vue-cli、Vuex、Vue-Router、Axios、Less
